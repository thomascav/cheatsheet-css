CSS -CheatSheet-

  • Text-style

    text-align
    Horizontal Alignment
    text-decoration
    Text Decorations (none underline line-through)
    font-family
    Font Face (Typeface) (Verdana, Arial, Helvetica, etc)
    font-size
    Font Size or Height
    font-weight
    Font Weight (Boldness)

  • Size and layouts

    width
    Width of HTML element
    height
    Height of HTML element
    margin
    space around an element, or distance between two elements
    margin-top
    Top Margin. Also try -bottom: -left: or -right:
    padding
    distance between an elements contents and its border
    padding-top
    Top Padding. Also try -bottom: -left: or -right:

  • Colours and borders

    color
    Element Colour
    background-color
    Background Colour of element
    background-image
    Background Colour of element
    border-color
    Border Colour of element
    border
    Width, style and colour of border defined together

  • CSS lists

    list-style
    Clear existing bullet types set by html list tags
